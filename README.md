# Home Care Application

Home-Care Application is a simple and practical tool designed to help manage daily health records and household inventory in one place. It allows users to easily save, view, and organize health information while also tracking available goods at home.

The application focuses on making inventory management easier by clearly showing which items are running low and which ones need to be purchased. By selecting items that need restocking, users can quickly see what to buy for the store, helping avoid missed or unnecessary purchases.

With an easy-to-use interface, the Home-Care Application helps users stay organized, monitor health records, and manage household supplies efficiently and stress-free.
